How to Install Elite Online on Your shared hosting?

  1. Create your domain or subdomain in your shared hosting
  2. Create an empty database in your shared hosting and remember this credentials:
    i. database name
    ii. database username
    iii. database password
  3. Upload the source code you have downloaded from CodeCanyon and unzip it.
    Note :- Exract your zip file and under that you find php code folder under that folder you will find php_code.zip file upload that file in your server,
    extract that file in your server
    Note :- .htaccess file is necessary , check that it exists , if not on the hidden file option
  4. Now navigate to the url where your project is located.
    ex. mydomain.com or subdomain.mydomain.com
  5. The installation screen should appear and guide you in the process.

Thats it! Now you are ready to run your first Elite Online : https://elitequiz.wrteam.in

    Elite Quiz Web App

    Elite Quiz Web Version is built with React JS v17.0.2 so you need to setup your Local Computer First.

  1. How to setup Web Version

    1. Install Node JS :
      2. Visit Node JS Official website : https://nodejs.dev/en/learn/how-to-install-nodejs/ for full installation guide.

  2. Where to set API URL (Admin)

    1. Go to Main Folder -> .env file open it Base URL
      (note:- do not change end of /api/ like set your admin panel url like this https://www.example.com/api/)

  3. How to run this project

    1. Unzip the downloaded code. After unzipping you will have Elite Quiz - Web Code folder.
      Open it in Visual Studio Code.
    2. Open VS Code teminal by typing CTRL+J in Windows/Linux, and for MacOS CMD+J and execute the command --> npm i
      This will take some time to download a few Packages so wait for a few minutes.
    3. After npm i finishes run this command --> npm start.
      This Command will Start the development mode.Check If everything is working fine then your are good to go ahead.
    ------------------------------
    Note:- If you already have purchased the App version of Elite Quiz and already created firebase project at that time then you can skip 4.1 step
    ------------------------------

  4. How to configure firebase configuration and authorised domain add.

    1. Create firebase project in your account



    2. Add web application to your firebase project
    3. Enter the App Name and Click on Register App
    4. How to configure firebase integration and domain in firebase console.
    5. Copy this Highlited Code and paste one by one as per label given in admin panel->web settings->settings
    6. This credentials should match with the credentials that you have set in admin panel. Otherwise it will not work properly.
    7. How to configure domain in firebase console
    8. You have successfully connected your Web application to your firebase project.

  5. How to enable firebase auth

    1. Go to firebase project.Click on authentication menu and goto sign-in method. Enable all the authentication method mention in below image
    2. If your are using google sign-in you need to enable OAuth APIs that you want to use using https://console.developers.google.com/ .Enable people api and make sure you've filled out all required fields in the console for OAuth consent screen. Otherwise, you may encounter APIException errors.
    3. After that Add your domain name without https:// OR http:// e.g elitequiz.wrteam.in
    4. Please add support email as shown in below image. Go to project settings -> general -> public settings

  6. How to enable cloud firestore

    1. Follow this steps in order to add cloud firestore

    2. Change security rules of cloud firestore in order to make your web more secure

  7. How to change Web Application name

    1. Change application name for your Web application. Go to Admin Panel > Profile and change the value of appName

  8. How to change logo, favicon, Firebase, footer data, quizplay images etc

    1. Go to Admin Panel -> Web Settings -> Settings.

      2. Seo meta tags

    2. Go to public -> index.html (Note:- make sure check whatever you set favicon in admin panel set here same with .png,.svg etc)

  9. How to set default language

    1. By default applicaiton language is English. If you want to change default language go to YOUR_APP_FOLDER > src > utils > config.js and add your respective language's code shown in below image. Get your language code from here https://developers.google.com/admin-sdk/directory/v1/languages.
      Make sure your language code is avaialble in https://developers.google.com/admin-sdk/directory/v1/languages.
    2. If your default language is not in src/locale/[language-code].json then create new file in same folder with [language-code].json. Add all the labels from en.json and convert label values in your respective language
    3. copy all the labels form en.json file in your [language-code].json file.

  10. How to add new language

    1. Go to your admin panel and enable language mode as shown in below image
    2. Go to your admin panel and add new language as shown in below image
    3. Add newly added language code in list as shown in below image. Go to src > utils > config.js Get your language code from here https://developers.google.com/admin-sdk/directory/v1/languages. Make sure your language code is avaialble in https://developers.google.com/admin-sdk/directory/v1/languages.
    4. Create new file in assets/languages/ with [language-code].json. Add all the labels from en.json and convert label values in your respective language
    5. copy all the labels form en.json file in your [language-code].json file.

  11. How to change application colors

    1. Add your colors in YOUR_APP_FOLDER/src/assets/scss/common/_color.scss and do not change the color variables name

  12. How to change application font

    1. Go to YOUR_APP_FOLDER/src/assets/scss/common/_font-size.scss and add your font as shown in image

  13. How to Add Slider

    1. Go to admin panel -> web settings -> slider

  14. How to Add Badges Images

    1. Go to admin panel -> Settings -> Badges Settings

  15. How to Change LTR to RTL

    1. Go to Admin Panel -> Web Settings -> Settings

  16. How set the Domain URL

    1. Go to package.json file and add your domain url in homepage variable. for ex, https://elitequiz.wrteam.in/

  17. How to deploy the Site

    1. After all the changes you are ready to deploy your Web applicaiton on Live Server.
    2. Open VS Code teminal by typing CTRL+J in Windows/Linux, and for MacOS CMD+J and execute the command --> npm run build
      This will take few seconds to build the live and the optimized version of the web application.
    3. After the process completion go to public > build folder and upload this files on your live server.
      (only upload build folder's files on server do not upload any other file on server)
    4. If you are facing any deployment related problems you can follow the official document : React App Deployment

  18. How to change Timer seconds

    1. Go to Admin Panel -> Settings -> System Utilities

  19. How to disable quiz mode features

    1. Go to Admin Panel -> Settings -> System Configuration

  20. How to Add Score

    1. Go to Admin Panel -> Settings -> System Utilities
